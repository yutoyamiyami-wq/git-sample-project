Git Sample Project
==================

Author: 富岡優友

git clone は、リモートリポジトリを初めて自分のPCにコピー（複製）するためのコマンド
使用例：git clone https://github.com/yutoyamiyami-wq/git-sample-project
使用される状況
・新しくプロジェクトに参加したとき
・別のPCで同じプロジェクトを作業したいとき
・オンライン上のソースコードを取得したいとき

git pull は、すでに clone 済みのローカルリポジトリを最新状態に更新するコマンド
使用例：git pull origin main
使用される状況
・チームメンバーが変更を加えた後
・作業を再開する前に最新状態を確認したいとき
・競合（コンフリクト）を避けるための事前更新

以下のURLにプロジェクトを公開した：
https://github.com/yutoyamiyami-wq/git-sample-project
